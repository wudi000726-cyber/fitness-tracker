# 🏋️ 情侣健身记录

一个专为情侣设计的健身记录网页应用，支持力量训练、有氧运动、体重追踪和打卡。

## 🚀 使用方法

1. 用浏览器打开 `index.html` 即可使用
2. 所有数据保存在浏览器 localStorage 中，无需联网

## ✨ 功能

### 📝 记录
- **打卡**：点击大按钮完成每日健身打卡
- **力量训练**：添加动作 → 填写每组重量×次数 → 保存
- **有氧运动**：记录跑步/骑行/游泳等，包含距离、时长、卡路里
- **体重记录**：追踪体重变化
- **模板加载**：从预设训练模板一键填充

### 📋 历史
- 按类型和日期范围筛选
- 展开查看每次训练详情
- 支持删除记录

### 📊 统计
- 总训练次数、打卡天数、连续运动天数、总训练量
- 体重变化趋势折线图
- 每周训练次数柱状图

### ⚔️ PK
- 两人本周数据并排对比
- 训练次数、打卡天数、连续天数、训练量
- 领先一方高亮显示

### 📑 模板
- 预设 3 个训练模板（推胸日、拉背日、腿部日）
- 支持自定义创建、编辑、删除模板
- 一键加载到记录页

### ⚙️ 设置
- 修改两人的名字
- 导出全部数据为 JSON 文件
- 从备份文件导入恢复数据
- 清除全部数据

## 🛠 技术

- 纯 HTML/CSS/JS，单文件应用
- Chart.js 图表库
- localStorage 本地存储
- 响应式设计，支持手机和电脑

## 📱 移动端使用

在手机浏览器打开 `index.html`，可添加到主屏幕作为 PWA 使用。
